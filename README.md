# 🛡️ BTM-SYS // Secure Interactive Portfolio v1.3

¡Bienvenido al centro de operaciones de mi portafolio profesional! Esta plataforma simula una terminal táctica de monitoreo de infraestructura y ciberseguridad, diseñada de manera limpia, eficiente y altamente interactiva.

🌐 **Despliegue en producción:** [btm-sys.netlify.app](https://btm-sys.netlify.app)

---

## 🛠️ Arquitectura y Componentes Técnicos

El proyecto está construido desde cero, cuidando el rendimiento, la semántica del código y la optimización de recursos, sumando más de **2,300 líneas de código estructurado**.

* **Estructura Semántica (HTML5):** Bloques organizados, accesibles y optimizados para SEO y maquetación limpia.
* **Diseño Cyberpunk / Terminal (CSS3):** Estilos oscuros con efectos de neón reactivos, animaciones analógicas secas y un sistema de rejilla (Grid/Flexbox) simétrico en 3 columnas para resoluciones de escritorio.
* **Lógica de Estados e Interactividad (Vanilla JavaScript):**
    * **Monitor del Sistema (Server Status):** Motor dinámico que calcula métricas fluctuantes de CPU, RAM y red.
    * **Simulador de Crisis de Energía:** Algoritmo probabilístico que genera ráfagas de chispazos eléctricos impredecibles y estados de desconexión completa (`OFFLINE`) con congelamiento lógico de datos para emular un falso contacto en hardware antiguo.
    * **Carga Segura (Preloader):** Sistema de inicialización de protocolos simulados mediante manipulación del DOM.
